## Version 2
---------

### Sorunlu özellikler 
> HTTPS hala sıkıntılıdır.

### Değişiklikler

* Yapı değişikliği ve güncelleme

### Scriptler

```
npm run start
```
> Kodu derleme ve çalıştırma

```
npm run build
```
> Kodu derleme ve derlenmiş dosyaları kaydetme

```
npm run rebuild
```
> Kodu yeniden derleme ve eski dosyaları silerek, güncelleme
